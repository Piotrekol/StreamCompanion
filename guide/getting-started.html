# Installation

StreamCompanion can be either installed or used as a portable application. Currently auto-updates are supported only for installed versions, while portable one will only notify you about new updates.

Installer: Grab latest setup exe from github releases pageopen in new window, and proceed with installation. Windows might complain about setup being unsigned - you can either bypass this window or compile it yourself using these stepsopen in new window.
Portable: Grab latest portable zip package from github releases pageopen in new window and unpack it in any non-system folder where you have read&write permissions.

WARNING

Do not run either osu! or StreamCompanion as administrator. Doing so will result in unexpected behavior (like SC not detecting any songs or ingame overlays not displaying anything)
